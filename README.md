## Command Signature

`php artisan burger:build`

## Parameters

- top-bun
- bottom-bun
- patty-count
- tomato
- cheese
- lettuce

`php artisan burger:build  --bottom-bun --patty-count=1 --tomato --cheese --lettuce --top-bun`
